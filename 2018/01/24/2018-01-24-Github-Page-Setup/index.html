Github Page Setup

2018-01-24
jekyll

github page使用的引擎jekyll，jekyll是静态网页生成器。
jekyll包含一些用markdown语法写的文件的文件夹；通过markdown和Liquid转换器生成一个网站。

安装jekyll

gem install bundler  
gem install jekyll

本地启动jekyll

打开git bash
进入jekyll项目的主目录

bundle install
jekyll serve --port 5000
